# Tic tac toe

Technology used


## Tech Stack

**Client:** nodejs, socket.io-client

**Server:** Node, Express, socket.io


## Installation

Install my-project with npm

```bash
    cd client && yarn
    cd server && yarn
```
 start server 

```bash
    yarn start
```
start as client

```bash
    yarn start
```
